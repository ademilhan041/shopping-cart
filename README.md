# Shopping Cart
## Project Details
This project has been developed by using Maven and Java8. jUnit4 has been used for testing.
You can build by executing command as in below:
```
mvn clean install
```

You can also run ```Main``` class to see result.

**Note:** 
* Only domain layer has been implemented based on DDD paradigm.
* The main class is ```ShoppingCartCalculator```.
* ```Campaign``` and ```Coupon``` have been designed as bounded context.
* ```Product``` and ```Category``` has been developed as simple as possible.
